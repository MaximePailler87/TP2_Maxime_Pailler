# TP2 : Tester des contenus dynamiques avec Selenium

Ce projet a pour but de vérifier le bon fonctionnement de plusieurs fonctionnalités du site _https://the-internet.herokuapp.com/_
On s'interesse ici aux fonctionnalités suivantes :
* Dynamic Controls : voir les programmes _dynamic_control_page.py_ et _tests_tp2_dc.py_
* Dynamic Loading : voir les programmes _dynamic_load_page.py_ et _tests_tp2_dl.py_
* Notification Message : voir les programmes _notification_message_page.py_ et _tests_tp2_nm.py_
* Infinite Scroll : voir les programmes _infinite_scroll_page.py_ et _tests_tp2_is.py_
